# Travel App (Flutter MVVM)

This project is a Flutter implementation of the Travel App UI using MVVM architecture.

## Structure

- `lib/models`: Data models (Destination)
- `lib/viewmodels`: Business logic and state management (HomeViewModel)
- `lib/views`: UI Screens (HomeScreen)
- `lib/widgets`: Reusable UI components

## How to Run

1. Ensure you have Flutter installed on your machine.
2. Copy the `flutter_app` folder to your local machine.
3. Open a terminal in the `flutter_app` directory.
4. Run `flutter pub get` to install dependencies.
5. Run `flutter run` to start the app on your emulator or device.

## Dependencies

- `provider`: For state management (MVVM)
- `google_fonts`: For typography
